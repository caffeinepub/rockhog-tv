# Specification

## Summary
**Goal:** Remove the Bacon Ops Arena experience from RockHog TV.

**Planned changes:**
- Remove Bacon Ops Arena from navigation/entry points so it is no longer accessible.
- Remove or disable any related UI routes/pages and backend endpoints/state tied specifically to Bacon Ops Arena.

**User-visible outcome:** Users will no longer see or be able to access Bacon Ops Arena within RockHog TV.
